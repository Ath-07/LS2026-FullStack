# Judgmental Password Strength Meter

A deliberately terrible UI component built for the Learner's Space 2026 Week 1 assignment.

This project reimagines the classic password strength meter as an overly judgmental assistant that criticizes users instead of helping them.

## Features

* Real-time password strength evaluation
* Sarcastic feedback messages
* Random color changes
* Input shake animation for weak passwords
* Fake show-password behavior

## Technologies Used

* HTML
* CSS
* JavaScript
